About This Website

This website was created as part of my web development learning journey. It demonstrates different layouts and components built using HTML5 and CSS3, including navigation menus, forms, banners, and styled content sections.

The purpose of this project is to practice structuring multipage websites, designing responsive layouts with flexbox, and building consistent page designs with headers, navigation, main content, and footers.

Every page highlights a specific assignment task, such as a pricing table, booking form, or embedded media. Together, these pages showcase my progress in web design and development.
